Input group

Easily extend form controls by adding text, buttons, or button groups on either side of textual inputs, custom selects, and custom file inputs.

Basic example

Place one add-on or button on either side of an input. You may also place one on both sides of an input. We do not support multiple form-controls in a single input group and labels must come outside the input group.

Sizing

Add the relative form sizing classes to the .input-group itself and contents within will automatically resize—no need for repeating the form control size classes on each element. Sizing on the individual input group elements isn’t supported.

Checkboxes and radios

Place any checkbox or radio option within an input group’s addon instead of text.

Multiple inputs

While multiple inputs are supported visually, validation styles are only available for input groups with a single input.

Multiple addons

Multiple add-ons are supported and can be mixed with checkbox and radio input versions.
